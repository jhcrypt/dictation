# Voice Commands

## Snippets
| Say | Action |
|-----|--------|
| "create snippet" | Start creating a new snippet (3-step voice flow) |
| "[trigger word]" | Expands to the saved snippet text |

## Text Editing
| Say | Action |
|-----|--------|
| "scratch that" | Deletes the last dictated text |
| "undo that" | Deletes the last dictated text |
| "delete that" | Deletes the last dictated text |

## Navigation
| Say | Action |
|-----|--------|
| "new line" / "next line" | Inserts a new line (Shift+Enter in chat apps) |
| "new paragraph" | Inserts two new lines |
| "tab" / "indent" | Inserts a tab |

## Clipboard
| Say | Action |
|-----|--------|
| "copy that" / "copy last" | Copies last dictated text to clipboard |
| "copy all" | Selects all and copies (Cmd+A, Cmd+C) |
| "paste" / "paste that" | Pastes clipboard (Cmd+V) |
| "select all" | Selects all text (Cmd+A) |

## Punctuation
| Say | Action |
|-----|--------|
| "period" / "full stop" | . |
| "comma" | , |
| "question mark" | ? |
| "exclamation point" / "exclamation mark" | ! |
| "colon" | : |
| "semicolon" | ; |
| "ellipsis" | ... |
| "dash" | — |
| "hyphen" | - |
| "open paren" | ( |
| "close paren" | ) |

## Hotkeys
| Shortcut | Action |
|----------|--------|
| Hold record key | Start recording |
| Release record key | Stop and transcribe |
| Escape | Cancel recording |
| Ctrl+Z | Scratch last dictation |
| Ctrl+, | Open settings |
